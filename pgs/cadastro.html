Cadastro

Nessa página o cliente irá realizar seu cadastro, onde será pedido o e-mail, cpf, endereço, idade, sexo...

Além de pedir o número de série para o cliente, para identificar no banco de dados o modelo da bike, que seria um complemento para o scanner.

Após finalizar o cadastro o cliente será redirecionado para a página do scanner para ecanear sua blicicleta e continuar com seu registro.
